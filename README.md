# 🧠 AI Focus & Habit Intelligence System

An AI-based system that evaluates a user’s focus and discipline using a memory game and daily habit analysis.

---

## 🚀 Features

- Memory-based focus game  
- Difficulty levels (Easy, Medium, Hard)  
- Habit analysis (study, sleep, phone, exercise)  
- AI-based scoring (Focus + Discipline)  
- Personalized suggestions  
- Graph visualization  

---

## 🧩 How It Works

1. Start the application  
2. Select difficulty  
3. Play focus game  
4. Answer habit questions  
5. View AI analysis and suggestions  

---

## ⚙️ Setup & Run

1. Install dependency:

pip install matplotlib

2. Run the project:

python main.py

---

## 🤖 AI Logic

Focus Score is calculated using:

Focus = (Game Score × 0.6) + (Discipline × 0.4)

The system analyzes user habits and generates personalized suggestions based on performance.

---

## 🎯 Purpose

To help students improve focus, discipline, and productivity using simple AI logic.
