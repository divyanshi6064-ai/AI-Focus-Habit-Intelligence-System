# 🧠 AI Focus & Habit Intelligence System

An AI-based interactive system that analyzes a user's focus, discipline, and daily habits using a combination of a cognitive game and intelligent rule-based analysis.

---

## 📌 Overview

This project is designed to help students improve their productivity and self-discipline.

It combines:
- A memory-based focus game
- A habit analysis system
- Rule-based AI logic for personalized suggestions

The system evaluates both mental focus and daily routine, then provides insights and improvement strategies.

---

## 🚀 Features

- Difficulty selection (Easy, Medium, Hard)
- Memory-based focus training game
- Habit analysis using sliders (study, sleep, phone usage, etc.)
- AI-based scoring system (Focus + Discipline)
- Personalized suggestions for improvement
- Performance visualization using graphs
- User-friendly GUI built with Tkinter

---

## 🧩 How It Works

1. Start the Application  
   - User clicks "Start"

2. Select Difficulty  
   - Easy → Basic number memory  
   - Medium/Hard → Advanced logic (future scope)

3. Play Focus Game  
   - A sequence is shown  
   - User recalls and enters it  
   - Score is calculated  

4. Answer Habit Questions  
   - Study hours  
   - Sleep hours  
   - Phone usage  
   - Exercise  
   - Daily activities  

5. AI Analysis  
   - Focus Score = Game performance  
   - Discipline Score = Habit analysis  
   - Combined using rule-based logic  

6. Final Output  
   - Scores displayed  
   - Suggestions generated  
   - Graph visualization

## 🤖 AI Logic Used

This project uses rule-based AI logic (no machine learning).

It combines:
- Game Score (Focus)
- Habit Score (Discipline)

Formula:

Focus Score = (Game Score × 0.6) + (Discipline × 0.4)

Based on score:
- High → Good discipline  
- Medium → Needs improvement  
- Low → Poor habits  

The system generates personalized suggestions accordingly.
