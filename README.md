# 🧠 AI Focus & Habit Intelligence System

## 📌 Overview

The **AI Focus & Habit Intelligence System** is a Python-based GUI application designed to analyze a user's focus and daily habits. It combines a cognitive memory game with lifestyle inputs to generate intelligent feedback and improvement suggestions.

This project aims to help users improve productivity, discipline, and self-awareness through simple and interactive analysis.

---

## 🚀 Features

* 🎮 Memory-based game to measure focus
* 📊 Habit analysis (study, sleep, phone usage, exercise)
* 🧮 Focus Score calculation
* 📈 Discipline Score evaluation
* 🤖 AI-based feedback (rule-based system)
* 💡 Personalized improvement suggestions
* 📉 Graph visualization using Matplotlib
* 🖥️ Interactive GUI using Tkinter

---

## 🧩 How It Works

1. User starts the application
2. Selects difficulty level
3. Plays the memory game
4. Enters habit-related inputs
5. System analyzes data
6. Displays:

   * Focus Score
   * Discipline Score
   * Feedback
   * Suggestions
   * Graph

---

## 🧠 AI Component

This project uses a **rule-based AI approach**.

The system:

* Analyzes user performance in the memory game
* Evaluates habit data (study, sleep, phone usage, exercise)
* Applies predefined conditions
* Generates adaptive feedback and suggestions

This simulates basic artificial intelligence by making decisions based on user behavior and performance.
